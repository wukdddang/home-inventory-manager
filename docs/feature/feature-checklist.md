# 기능 체크리스트 (Home Inventory Manager)

**버전**: v2.5 — UUID 확정 + Purchase 파생값 제거 (2026-03-28)

**v2.5 변경**:
- Purchase에서 `quantity`(파생값), `totalPrice`(파생값) 제거 — 구매 기록 기능 설명 갱신

**v2.4 변경**:
- 알림 설정(NotificationPreference) 마스터 토글 3종 추가
- 장보기 완료 트랜잭션 API 스펙 정의

**v2.3 변경**:
- 거점 유형 정의(HouseholdKindDefinition) CRUD 추가

**v2.2 변경**:
- 거점 초대(HouseholdInvitation) 기능 추가
- 거점 멤버 역할 3단계 확장 (admin/editor/viewer)

**v2.1 변경**:
- 카테고리, 단위, 상품 → 거점별 카탈로그
- "다른 거점 카탈로그 가져오기" 기능 추가
- 알림 설정(NotificationPreference) CRUD 추가

**v2 변경**:
- 소비 기록, 폐기 기록 → 재고 변경 이력 섹션으로 통합
- 장보기 리스트 섹션 제거, 장보기 항목 섹션에 병합
- 장보기 제안 기능 추가
- 구매 → 재고 자동 반영 기능 추가
- 소비·폐기 처리 기능 명시

엔티티 목록·관계는 [er-diagram.md v2](../design/v2/er-diagram.md), 필드 상세는 [entity-logical-design.md v2](../design/v2/entity-logical-design.md)와 동기화합니다.
