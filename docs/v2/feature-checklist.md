# 기능 체크리스트 v2 (Home Inventory Manager)

**버전**: v2.1 — 카탈로그 거점별 분리 + 알림 설정 테이블 추가 (2026-03-26)

**v2.1 변경**:
- 카테고리, 단위, 상품 → 거점별 카탈로그
- "다른 거점 카탈로그 가져오기" 기능 추가
- 알림 설정(NotificationPreference) CRUD 추가

**v2 변경**:
- 소비 기록, 폐기 기록 → 재고 변경 이력 섹션으로 통합
- 장보기 리스트 섹션 제거, 장보기 항목 섹션에 병합
- 장보기 제안 기능 추가
- 구매 → 재고 자동 반영 기능 추가
- 소비·폐기 처리 기능 명시

엔티티 목록·관계는 [er-diagram.md v2](./er-diagram.md), 필드 상세는 [entity-logical-design.md v2](./entity-logical-design.md)와 동기화합니다.
**v1 원본**: [v1/feature-checklist.md](../v1/feature-checklist.md)
