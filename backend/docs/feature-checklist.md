# 엔티티 명세 (Home Inventory Manager)

실제 **엔티티**와 연결된 기능만 정리한 명세입니다.  
완료한 항목은 `- [x]`로 표시해 관리하세요.

---

## User (사용자)

- [ ] 사용자는_회원가입을_한다
- [ ] 사용자는_내_정보를_조회한다
- [ ] 사용자는_비밀번호를_변경한다

---

## Household (가족 / 공유 그룹)

- [ ] 사용자는_가족·공유_그룹을_생성한다
- [ ] 사용자는_가족·공유_그룹_목록을_조회한다
- [ ] 사용자는_가족·공유_그룹_정보를_수정한다
- [ ] 사용자는_가족·공유_그룹_멤버를_추가한다
- [ ] 사용자는_가족·공유_그룹_멤버를_제거한다
- [ ] 사용자는_가족·공유_그룹을_삭제한다

---

- [ ] 사용자는_카테고리_목록을_조회한다
- [ ] 사용자는_카테고리를_수정한다
- [ ] 사용자는_카테고리를_삭제한다

---

## PurchaseBatch (유통기한 로트)

> **로트(lot)**: 한 번에 구매한 같은 품목 묶음. 같은 유통기한을 공유하는 단위.  
> 예: 우유 2팩을 3월 1일에 구매하고 유통기한이 3월 10일이면, 그 2팩이 한 로트.

- [ ] 사용자는_유통기한_로트_목록을_조회한다
- [ ] 사용자는_유통기한_임박_목록을_조회한다
- [ ] 사용자는_이미_만료된_목록을_조회한다
