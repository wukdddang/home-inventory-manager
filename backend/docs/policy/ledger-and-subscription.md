# 가계부·구독 — 범위 외 안내

본 프로젝트(**Home Inventory Manager**)의 핵심 목표는 **개인/가정 재고·물품 관리**입니다.

**가계부**(잔고, 입출금, 수입/지출 분류, 예정 수입 등), **구독**(반복 결제, 다음 결제일 알림), **예산**(월별·카테고리별 예산 설정) 기능은 본 시스템 범위에 두지 않습니다.

- **권장**: 가계부·구독·예산이 필요하면 **별도 프로젝트**로 설계·개발한 뒤, 필요 시 API 연동 등으로 붙이는 방향을 권장합니다.
- 기존에 본 문서에 정리했던 엔티티(Account, AccountTransaction, RecurringIncome, TransactionCategory, RecurringExpense, Subscription, Budget)는 ERD·개념/논리 설계에서 제거되었습니다.

이전 상세 정리(가계부 확장 방식, 구독 데이터 수집 방법 등)가 필요하면 git 히스토리에서 확인할 수 있습니다.
